# cometkim's Homebrew Tap

## How do I install these formulae?

`brew install cometkim/tap/<formula>`

Or `brew tap cometkim/tap` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "cometkim/tap"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
